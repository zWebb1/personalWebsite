Project #2

C:\Zachary\Webb\SearchNear> "Search Near" is an app made in Android Studio using Kotlin. This was a final project made for mobile application development. We used Google's API call "Search for nearby places" to help users find nearby restaurants, grocery stores, and gas stations. The first thing the users see is a dark mode screen with an option to switch to light mode if they choose. They are presented the three searching options and can specify the radius within which they would like to search. Once they click on the search button, the app takes their location and the parameters they entered and navigates to a different screen, which is a map that displays their current location along with all the locations of the selected industries. The app also includes a back button that allows users to go back and perform a new search.

C:\Zachary\Webb\SearchNear> Technologies: Android Studio, Kotlin, Google APIs
